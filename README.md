# whispers

A first-person 3D psychological horror adventure about a forest that collects voices.

You wake in the woods with no memory of how you got here. Something with a face that's almost human offers to help you escape. The dead whisper warnings in the dark — but they don't know they're dead. They think they escaped. They're trying to save you from a trap they already fell into.

The only way out, the thing with the face says, is to bring someone else in.

## Status

Vertical slice in development. Target playtime: 20–30 minutes.

## Stack

- **Engine:** Godot 4.x (GDScript)
- **Assets:** Free packs (Kenney.nl, Sketchfab CC0, itch.io, Freesound.org)
